Exercice : Formulaire Inscription

Consignes :

  • 2 Gérer les erreurs avec un fichier JS (Si possible)
  • 3 Assurez-vous que le formulaire est accessible aux personnes utilisant des technologies d'assistance, en testant son accessibilité à l'aide d'outils tels que Wave ou Axe.
